## 💡 part2-4 상세페이지 200만개 만들기 (URL 파라미터)

### 🔹 라우부트스트랩을 설치 시 사용하기 좋은것

- mt mb ml 등 마진과 패딩을 클래스명 만으로 줄 수 있다

### 🔹 상세페이지

- path에 적은 경로 에 /:id를 붙여주면됨

```javascript
//id에는 정규식을 사용할 수 있다
*이라는것도 있음 이건 여러개 반복
"/list/:id";
```

### 🔹 파라미터

- `$route.params`는 파라미터 자리에 기입된것을 다 볼 수 있다

### 🔹 404페이지

- `/:anything(.*)` 기본형식
- 모든 문자라는 정규식
- 이건 무조건 제일 마지막
